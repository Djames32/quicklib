FAQ

How do I download an e-book for offline reading?

Open a book’s page and click Download. Once it’s saved, you can access it anytime from your Library/Downloads section.

What file types do you support?

We support common e-book formats like PDF and EPUB (and more can be added later). If a book isn’t supported, you’ll still be able to view its details and choose an alternate version if available.

Do I need an account to start reading?

No—Free users can browse and read online right away. Creating an account lets you save favorites, track reading lists, and sync your library across devices.
